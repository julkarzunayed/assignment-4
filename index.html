Question-1: What are the different ways to select an element in the DOM?

  1. 01. Using "getElementById"
  2. 02.Using "getElementsByClassName" and looping on them.
  3. 03. Using "querySelector" for the child of a parentNode.
  4. 04. Using "querySelectorAll" and lopping on them.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

They all allow us to read and update the content of HTML elements.
innerHTML: is used to get access of inner content with element tag name.
textContent: is used to get access of inner content without the tag names and returns all text as it is rendered on screen including all comment .
innerText: is used to get access of inner text only ignoring element tag names and also does not include the text that is hidden with CSS style.
